# Python Port Scann Description.
A simple port scanner built using Python that detects open ports on a target system.

##  Features.
- Scans ports 1–100
- Multithreading for speed
- Domain to IP conversion.

 ## How to Run 
python scanner.py

## Concepts Used...
- Sockets
- Networking.
- Multithreading
- Cybersecurity basics

## Web Interface...
1. Install Flask: `pip install flask`
2. Run server: `python app.py`
3. Open browser: `http://localhost:5000`

## CLI Usage..
Run `python scanner.py` for command-line scanning of ports 1–100..
